Základy JS - opakování

1: Co je JavaScript a k jakým účelům se používá?

Doplň vhodné odpovědi a využij k tomu nečíslovaného seznamu.

Použij v textu aspoň 3 internetové odkazy na zajímavé příklady využití JS!

  • JavaScript je multiplatformní, objektově orientovaný, událostmi řízený skriptovací jazyk, jehož autorem je Brendan Eich z někdejší společnosti Netscape.
  • Používá se pro interaktivitu, >dynamickéfunkce na webových stránkách.
  • Syntaxe JavaScriptu je podobná jiným programovacím jazykům, jako je C, C++, a Java, což usnadňuje učení a používání.
  • JavaScript umožňuje manipulaci s webovými stránkami a dokumenty, včetně změny obsahu, přidávání událostí a reakcí na uživatelské interakce.
  • JavaScript se využívá i na straně serveru. První implementací JavaScriptu na straně serveru byl LiveWire firmy Netscape vydaný roku 1996.
  • JavaScript je podporován většinou moderních prohlížečů a není omezen na konkrétní platformu, což umožňuje psát kód, který běží na různých zařízeních a prohlížečích.

2: Jaká pravidla platí pro psaní kódu v JavaScriptu?

Napiš aspoň pět zásad ve podobě číslovaného seznamu.

Použij angličtinu!

  1. Use consistent and clear indentation to make your code more readable. A common convention is to use 2 or 4 spaces for each level of indentation.
  2. Use meaningful names for variables and functions that describe their purpose. This makes your code self-documenting and easier to understand.
  3. Add comments to explain complex or non-obvious parts of your code. Well-placed comments help other developers (and your future self) understand your code.
  4. Minimize the use of global variables to prevent naming conflicts and maintain data encapsulation. Use local or function-scoped variables whenever possible.
  5. Follow a consistent coding style, whether it's your own, your team's, or a recognized style guide like "JavaScript Standard Style" or "Airbnb JavaScript Style Guide." This ensures uniformity and readability across your codebase.

4: Jak je možné v JavaScriptu deklarovat proměnné?

Uveďte na příkladech různé datové typy, které je možné využívat v JS.

Doplňte chybějící kód/text podle vzoru úvodních dvou řádků!

let x = 0; // s přiřazením hodnoty
let x; // bez přiřazení hodnoty
let x = null; // s přiřazením prázdné (žádné) hodnoty
let cele = 3; // celé číslo (typ number)
let realne = 2.5; // reálné číslo (typ number)
let desetinne = 10e1; // zápis desetinného čísla pomocí exponentu
let text = "DOBRÝ DEN"; // typ string
let prestupny = false; // proměnná typu boolean, nabývá pouze hodnot true nebo false

6: Která popup okna lze využít v JavaScriptu?

K uvedeným příkladům připište do poznámky, co se stane na webové stránce.

Na konci doplňte aspoň dvě varování, proč bychom se měli vyhýbat použití popup oken v reálných webových aplikacích.

alert()

alert('Špatná odpověd');
// Nahoře se zobrazí vyskakovací okno se zprávou "Špatná odpověd"

prompt()

let odpoved = prompt('Odpoved:');
// Do proměnné odpoved vloží odpoved zadané uživatelem

confirm()

if (confirm('Chceš se přemistit na Wikipedii'))location.href='https://cs.wikipedia.org';
// Nahoře se zobrazí vyskakovací okno, které se uživatele ptá, jestli chce přesměrovat

Varování:

  1. Rušivost pro uživatele: Pop-up okna mohou být rušivá a nepříjemná pro uživatele
  2. Zablokování obsahu: Pop-up okna mohou zablokovat obsah na hlavní stránce webové
  3. Prohlížeč může vyskakovací okna blokovat.
